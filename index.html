An application container platform built on kubernetes

Nav's Application Infrastructure Service

NAIS is an application platform built to increase development speed by providing our developers with the best possible tools to develop, test and run their applications.

Naisible

Naisible is a set of ansible playbooks that build and configure a kubernetes cluster on linux nodes

Naisd

Naisd is the golden path to continous delivery on the nais platform. It provides developers with a simple API to deploy their applications based on a minimalistic deployment yaml.

Learn more

Rook

Rook orchestrates the setup of a Ceph storage cluster in the Nais environment
With Ceph running in the cluster, we've made it easy to dynamically provision volumes, satisfying the demand for persistent storage.
